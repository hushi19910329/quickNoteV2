# 快捷笔记软件功能点设计

## 1. 产品目标与设计原则
- **目标**：构建一个轻量、始终可达、低打扰的桌面悬浮便签工具，覆盖“快速记录 -> 组织管理 -> 提醒执行”闭环。
- **核心原则**：
  - 少按钮、强交互：通过悬停、右键菜单、快捷键完成常用操作。
  - 内容优先：按钮高度控制在约 25px，边框与控件尽量轻量，最大空间给笔记正文。
  - Windows 浅色风格：整体采用浅色配色，保持干净、清爽、可读。
  - 快速切换：新建便签、颜色切换、标签筛选、展开/收起都在 1~2 步内完成。

## 2. 信息架构与主界面布局

## 2.1 主界面分区
- **顶栏（约 25px）**
  - 左侧：应用 Emoji 图标 + 当前视图标题（如“全部便签”）。
  - 右侧：Pin、透明度、最小化、设置入口（图标化，低显性）。
- **侧栏（可折叠）**
  - 快速过滤：全部、待办、提醒中、已归档。
  - 标签区：按标签筛选便签。
- **便签列表区**
  - 卡片式列表，仅展示标题、摘要、标签、提醒状态。
  - 支持仅列表模式（收起编辑区时使用）。
- **编辑区（可展开/收起）**
  - 富文本编辑（文字 + 图片混排，图片仅展示与简单排版）。
  - 高亮工具、颜色/标签快速设置、提醒设置入口。

## 2.2 交互状态
- **展开模式**：列表 + 编辑并存，用于深度编辑。
- **收起模式**：仅显示便签标题/待办列表，用于低干扰查看。
- **悬浮贴边模式**：拖拽到屏幕右边缘后自动吸附，鼠标靠近时展开。

## 3. 功能清单（含内容、布局、交互）

## 3.1 窗口与悬浮能力
- **功能内容**：
  - 桌面悬浮、最小化、最大化、恢复。
  - 右侧悬停靠边（自动贴边 + 自动展开）。
  - Pin 置顶模式（始终在最上层）。
  - 半透明显示与透明度可调。
- **布局**：
  - 顶栏提供 Pin 与透明度入口（图标/滑杆弹层）。
  - 状态在顶栏以小图标显示（如置顶已启用）。
- **交互**：
  - 拖拽窗口到右边缘触发贴边；移开鼠标自动缩回。
  - 快捷键切换 Pin/透明度（例如 `Ctrl+Shift+P`）。
  - 透明度支持滚轮微调与设置面板精确输入。

## 3.2 便签创建与快速操作
- **功能内容**：
  - 一键新增便签。
  - 快速切换便签颜色（用于视觉分组）。
  - 便签 Emoji 图标（每条便签可独立设置）。
- **布局**：
  - 顶栏 `+` 按钮；便签卡片右上角提供颜色点/Emoji 入口。
  - 右键菜单提供“复制、归档、删除、设提醒”。
- **交互**：
  - `Ctrl+N` 新建。
  - 颜色与图标通过弹出面板快速选择，不进入复杂设置页。
  - 默认自动保存，不需要手动保存按钮。

## 3.3 富文本与图片混排编辑
- **功能内容**：
  - 支持普通文本、标题、列表、引用、文本高亮。
  - 支持粘贴图片并插入正文；图片可缩放到预设尺寸（小/中/大）。
  - 图片不做复杂编辑，但可调整对齐与前后间距。
- **布局**：
  - 编辑区顶部为轻量工具条（高亮、列表、插图）。
  - 文本区域占主要空间；工具条在无焦点时弱化显示。
- **交互**：
  - `Ctrl+V` 直接粘贴图片到光标位置。
  - 选中文本后浮动工具条显示高亮颜色选择。
  - 拖动图片边框角点切换展示尺寸档位。

### 3.3.1 图片交互增强（编辑体验）
- **点击选中 + 工具条**
  - 单击图片后显示图片工具条：放大、缩小、原始尺寸、适应宽度、预设尺寸（S/M/L）。
- **缩放交互**
  - `Ctrl + 鼠标滚轮` 对选中图片进行缩放。
  - `Alt + 鼠标拖拽` 可快速调整图片宽度（拖拽缩放）。
- **双击预览**
  - 双击图片弹出预览窗口，支持滚轮缩放与拖拽平移。
  - 预览窗口支持上一张/下一张（图集浏览）。
- **右键菜单操作**
  - 放大/缩小、原始大小、适应宽度、S/M/L 尺寸切换。
  - 左/中/右对齐、上下间距调节、添加图片说明（caption）。
  - 复制图片路径、在资源管理器打开、删除图片。

## 3.4 语音转写（硅基流动接口）
- **功能内容**：
  - 支持录音转文字并插入当前便签。
  - 支持“追加到末尾”与“插入到光标处”两种写入策略。
  - 支持转写失败重试与错误提示。
- **布局**：
  - 编辑区中提供“语音输入”按钮（麦克风 Emoji）。
  - 录音状态条显示时长、上传中、转写中状态。
- **交互**：
  - 点击开始录音，再次点击结束并自动上传转写。
  - 转写结果可一键“插入/替换/取消”。
  - 首次使用引导用户配置 API Key 与模型参数。

## 3.5 标签系统
- **功能内容**：
  - 创建、编辑、删除标签。
  - 单便签支持多标签。
  - 标签颜色与排序可配置。
- **布局**：
  - 侧栏展示标签列表与计数。
  - 便签编辑区标题下方显示已选标签 Chips。
- **交互**：
  - 输入 `#` 触发标签联想并快速附加。
  - 支持“按标签筛选 + 关键词搜索”组合过滤。

## 3.6 提醒与待办
- **功能内容**：
  - 为便签设置单次或重复提醒。
  - 待办项（checkbox）与到期状态跟踪。
  - 提醒触发时弹窗 + 声音（可关闭）。
- **布局**：
  - 便签卡片显示提醒时间与状态标记。
  - 设置提醒采用轻量弹窗（日期、时间、重复规则）。
- **交互**：
  - 到点本地通知，点击通知可定位到对应便签。
  - 支持稍后提醒（5 分钟/15 分钟/1 小时）。

## 3.7 收起/展开与低打扰模式
- **功能内容**：
  - 一键收起编辑区，仅显示列表（标题/待办）。
  - 悬浮显示时尽量不抢焦点，不遮挡主要工作内容。
- **布局**：
  - 顶栏显示收起/展开开关。
  - 收起后列表占满窗口，卡片信息密度提高。
- **交互**：
  - 双击便签卡片展开详情并进入编辑。
  - Esc 快速收起当前编辑区。

## 3.8 搜索与过滤
- **功能内容**：
  - 全文检索（标题、正文、标签）。
  - 过滤条件：颜色、标签、是否提醒、是否归档。
- **布局**：
  - 顶栏搜索框 + 侧栏过滤项。
- **交互**：
  - 输入即搜索，结果高亮匹配文本。
  - 保存常用过滤视图（如“今日到期”）。

## 3.9 数据管理与可靠性
- **功能内容**：
  - 本地 SQLite 持久化。
  - 自动保存、崩溃恢复、软删除回收站。
  - 导出/导入（Markdown + 图片资源包）。
- **布局**：
  - 设置页提供数据目录、备份、导入导出入口。
- **交互**：
  - 默认每次编辑节流保存（如 800ms）。
  - 启动时自动检测未完成写入并恢复草稿。

## 3.10 设置与主题
- **功能内容**：
  - 窗口行为设置（置顶、贴边、开机自启）。
  - 透明度与字体大小。
  - 硅基流动 API 设置（Base URL、API Key、模型）。
- **布局**：
  - 设置分为“外观、行为、提醒、语音、数据”5 个分组。
- **交互**：
  - 配置项即时生效，保留“恢复默认”。

## 4. 关键用户故事（User Stories）
- **US-01 快速记录**
  - 作为经常开会的用户，我希望按下快捷键后立即打开悬浮便签并开始输入，这样我能在 3 秒内记下临时想法而不打断当前工作。
- **US-02 语音转文字**
  - 作为不方便打字的用户，我希望点击麦克风直接录音并自动转写成文本插入便签，这样我可以快速记录灵感或会议纪要。
- **US-03 提醒闭环**
  - 作为任务驱动型用户，我希望给便签设置提醒时间并在到点收到通知，这样我不会遗漏待办事项。
- **US-04 标签检索**
  - 作为管理多项目笔记的用户，我希望通过标签和关键词组合筛选目标便签，这样我能在几秒内定位历史记录。
- **US-05 低打扰悬浮**
  - 作为需要长期专注的软件开发者，我希望便签能贴边悬浮并在鼠标靠近时展开，这样我能随手查看笔记又不遮挡代码窗口。

## 5. MVP 与迭代建议
- **MVP（第一阶段）**
  - 悬浮窗口 + 贴边 + Pin + 透明度。
  - 文本/图片混排编辑 + 高亮。
  - 标签系统 + 搜索。
  - 定时提醒 + 本地通知。
  - SQLite 持久化 + 自动保存。
- **第二阶段**
  - 语音转写（硅基流动接入）。
  - 数据导入导出。
  - 列表收起模式与更细颗粒交互优化。
- **第三阶段**
  - 多窗口/多桌面适配。
  - 插件式扩展（AI 摘要、智能标签建议）。
