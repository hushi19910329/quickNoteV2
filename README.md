# QuickNote V2（快捷便签式笔记）

一个基于 **PySide6** 的 Windows 浅色风格桌面悬浮笔记软件设计稿项目。  
当前版本以“产品功能规划 + 架构设计 + 数据库设计”为主，为后续编码实现提供清晰蓝图。

## 1. 核心特性（规划）
- 桌面悬浮、右侧贴边、最小化/最大化、Pin 置顶。
- 透明度可调、低打扰显示。
- 富文本编辑（文本高亮）+ 图片粘贴混排。
- 标签系统、颜色快速切换、便签 Emoji 图标。
- 定时提醒与待办追踪。
- 语音转写（硅基流动 API）。

## 2. 文档说明
- [功能点设计.md](./功能点设计.md)：完整功能清单、布局与交互设计、User Stories。
- [架构设计.md](./架构设计.md)：目录结构、分层设计、类和函数接口、关键流程。
- [数据库设计.md](./数据库设计.md)：SQLite 表结构、索引、初始化 SQL、迁移策略。

## 3. 运行方式
本项目要求通过 Python 方式运行，不编译为 EXE。

```bash
python main.py
```

> 说明：当前仓库为设计阶段，后续按架构文档补齐 `main.py` 与 `src/` 实现代码即可直接运行。

## 4. 开发环境建议
- Python 3.11+
- Windows 10/11
- 建议使用虚拟环境

```bash
python -m venv .venv
.venv\\Scripts\\activate
pip install -r requirements.txt
```

## 5. 里程碑建议
- M1：实现主窗口、便签 CRUD、自动保存、标签与搜索。
- M2：实现提醒调度与本地通知。

## 6. Git 版本管理要求
- 代码通过 Git 管理并推送到远程仓库：`https://github.com/hushi19910329/quickNoteV2`
- 建议分支策略：
  - `main`：稳定可发布版本
  - `feature/*`：功能开发分支
  - `fix/*`：缺陷修复分支
- 建议提交规范：
  - `feat:` 新功能
  - `fix:` 缺陷修复
  - `docs:` 文档更新
